# ConnectX
Open source real-time multiplayer strategy game where players connect X pieces on a dynamic board. Fully configurable, competitive, and built for skill and educational purposes.

### Development
Start both the client and server in watch mode:
```bash
pnpm dev
```

- Client: `http://localhost:5173`
- Server: `http://localhost:4000`

### Production Build (Docker)
To test the production build locally:
```bash
pnpm docker:prod
```

See `docs/development.md` for more details.
